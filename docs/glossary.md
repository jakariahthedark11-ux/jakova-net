# Glossary

**BIOS (in this repo)**  
The model layer acting like firmware: boots the system, helps mutate state, does not own the universe.

**Local-first**  
Architecture where core state and decision-making can live entirely on the user’s hardware.

**Universe**  
A particular JSON state file describing a cosmos, lab, factory, or any other world.

**Engine**  
A runtime (Python, Unity, etc.) that reads the JSON and renders or acts on it.

**EXAWATTS**  
Work-class and power-routing scheme for personal multi-AI operating systems. See the EXAWATTS Zenodo paper.

**EBNT**  
Evans-Barboza Network Theory – cross-scale framework for network structure in natural systems.

**FVE**  
Fractal Voyager Engine – the cosmos viewer demo for JAKOVA-NET.
